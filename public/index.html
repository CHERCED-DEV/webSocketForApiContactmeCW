Endpoints

La API de Contacto tiene los siguientes endpoints:

  • GET /api/contactMe: Devuelve una lista de todos los mensajes enviados.
  • POST /api/contactMe: Envía un nuevo mensaje de contacto.

GET /api/contactMe

El endpoint GET /api/contactMe devuelve una lista de todos los mensajes enviados a través de la API. Los mensajes se devuelven en formato JSON.

Parámetros de entrada

Este endpoint no acepta parámetros de entrada.

Respuesta

Este endpoint devuelve una lista de objetos JSON que representan los mensajes enviados a través de la API. Cada objeto tiene los siguientes campos:

  • name: El nombre del remitente.
  • email: La dirección de correo electrónico del remitente.
  • phone: El número de teléfono del remitente.
  • message: El mensaje enviado por el remitente.
  • createdAt: La fecha y hora en que se envió el mensaje.

POST /api/contactMe

El endpoint POST /api/contactMe permite a los usuarios enviar un nuevo mensaje de contacto a través de la API. Los mensajes se guardan en una base de datos MongoDB y se transmiten a través de un socket WebSocket a los clientes conectados.

Parámetros de entrada

Este endpoint acepta los siguientes parámetros de entrada en formato JSON:

  • name: El nombre del remitente.
  • email: La dirección de correo electrónico del remitente.
  • phone: El número de teléfono del remitente.
  • message: El mensaje enviado por el remitente.

Respuesta

Este endpoint devuelve un objeto JSON con el siguiente campo:

  • message: Un mensaje que indica que el mensaje se envió correctamente.
